# AI Tools Used & Assistance

For this project, I used **Google Deepmind Antigravity (powered by Gemini)** as my AI coding assistant. Since I already knew exactly how I wanted the system to work, I designed the core architecture, the database schema, and all the role-based access rules myself. I basically just used the AI to speed up the tedious parts of development:

- **Writing Boilerplate:** I had the AI quickly generate the basic structure for my Next.js components, Tailwind CSS styling, and Express routes based on my own designs.
- **Setting up CI/CD:** I used it to draft the initial GitHub Actions workflows for testing and linting, which I then tweaked and integrated with Vercel.
- **Generating Diagrams:** Instead of drawing diagrams by hand, I just fed my database logic into the AI and had it write the Mermaid.js and PlantUML code to generate my ERD and Architecture diagrams.
- **Debugging:** Whenever I hit a weird routing bug or a complex state issue, I used it like a pair-programmer to bounce ideas off of so I could find the fix faster.
